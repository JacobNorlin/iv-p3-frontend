Usage

Click on Countries or Demographics to get an ambient visualization of the the reach and like data for either countries or demographics. The parallell coordinates, table, and ambient visualisation all brush. This means you can filter the table and visualisation with the parallel coordinates, you can also use the table to highlight things in the parallell coordinates and visualisation. The ambient visualisation is not interactive in itself. The slider underneath the buttons is a timeline and can be used to select data from any date.

You can zoom the ambient visualisation with "Z" and "X" if you need to resize it.

Posts and Generic are presented as parallell coordinates with a data table. This is to be able to view more specific information about the facebook group.

Mappings

  • Blue line: Increase in reach
  • Red line: Decrease in reach
  • Speed: Amount of increase/decrease
  • Direction: Counterclockwise = decrease in reach, clockwise = increase in reach
  • Distance from center: Total weekly reach(lower = closer)
  • Size of ball: lifetime total likes
  • Shockwaves: Rate scales with gained like in the last week

Detailed data

For generic data color maps to daily engaged users where red is low, and blue is high.

For post data color scales with the type of the post.

The table is linked to the parallell coordinates. Hovering a row in the table will highlight the corresponding line in the parallell coordinates.
